## 关于博客

先来说说我这个博客，上次更新时间是2022年3月份，距今仅仅过去了不到4年，但是好像来到了两个世界。
2022年底 GPT3.5 版本发布并爆火，生成式大模型开始破圈进入大众的视野。
接下来几年 AI 技术飞速发展，更多的上下文，更快的生成速度，更好的理解能力，再到多模态大模型，MCP 协议, Agent 技术，AI 应用层出不穷，AI逐渐渗透到各行各业。

具体到软件开发领域，2024 年下半年 cursor 开始爆火，2025 年上半年基于命令行的 claude code 开始爆火，
越来越多的开发者开始使用 AI 辅助编程工具，AI 生成代码已经成为日常工作的一部分。

以我这个博客为例，我把博客框架从旧的 hogo 迁移到达 astro，并增加了如下功能:

1. 部署到 cloudflare, 并支持自动化部署
2. 增加显示每个 tag 下有多少个文章
3. 增加 Giscus 评论系统
4. 修复 edit post 功能，增加一个 post:new 命令行功能

我并不是 code agent 的专家，但是这些功能我一个下午就完成了。有几个功能我只输入一个提示词，code agent 就帮我完成好了。
如果是 2022 年的时候，这几个功能，我可能要两三天，因为 asrot，Gisus，tailwindcss 等很多细节我也不了解，
我需要查阅文档，可能做这方面工作和技术调研就需要一个下午。
而在2025年的今天，我一个下午就完成了所有工作，效率提高了好几倍。

所以2026年，我会继续 all in AI，借助AI，希望能做出一些不错的产品。

## 2025 年终总结

2025年，对我而言是一个多事之秋。

工作上我来到新公司已经三年了，从之前懵懂的不熟悉公司的业务，不熟悉技术框架，到现在也渐渐能够独挡一面。
受政治市场等因素影响，公司裁员今年裁员 3分之2，2000 人只剩下大几百人了，只是我运气不错，我的部门没有受到很大影响。
我司工作氛围还不错，人员流动挺低的，有很多员工都待了大几年甚至十多年，但可悲的是，员工选择了公司，公司却养不了这么多员工。
在裁员那几天，大家的心态真的受到很大的影响。
有些部门整个部门都裁掉了，很难想象之前管理几百人的高 title 领导，裁掉之后还好不好找到合适的工作？
总体而言，我还比较喜欢我这个公司和现在的这个部门。业务有一定的复杂度，技术框架抽象的很好，工作的深度和广度都不错，可以长期干下去。

第二件是就是我家宝宝4岁呢，开始上幼儿园呢。他已经开始有自己的想法，他会牵着我的手问，爸爸，这个标志是什么？孙悟空和猪八戒谁厉害呢？
他也会抢夺我的手机，说爸爸给我看一下，我想看动画片,我要玩游戏。
我意识到对宝宝的教育工作，我需要花费更多时间.
但他们的时代不一样了，他们有更多的诱惑，同时也面临着更多的竞争，比如包括AI方面的。
我的宝宝有一些我的优点，如果比较爱看书， 较强的好奇心，同情心，我希望能好好的教育他，让他成为一个积极的，自信的，丰富有趣的人。

![宝宝 4 岁了](@/assets/images/2026/bayb-4-years-old.jpg)

第三件事情就是我弟弟得了脑出血，而且出血量比较大，有50毫升，还好送医院及时，否则的话可能人都不在了。
我弟弟生这个病有自己的原因，也有了大环境的原因。
自己的原因就是比较胖，不注重锻炼，肥胖就容易有各种各样的慢性病。
大环境的原因就是因为疫情，他们公司的工资拖欠了好几个月没有发，然后上有老下有小，家里的开销比较大，所以他的压力也很大。
我弟弟才40岁，那一个晚上的情形，我还历历在目。
因为他脑功能受损，所以鼻子嘴巴里面都充满了体液，他已经不能自己自主呼吸。
然后医生给他插肺管是比较急，就导致鼻腔嘴巴里面都渗出血来。
我不清楚我弟当时是否还有自主意识，但他肯定很难受，他还是条件反射的去扯这个肺管。
看着我弟难受的样子，我的眼泪忍不住流下来。
把我弟媳也被这个突然的变故吓坏了，有点不知所措。
一个好好的家庭，不但没有了主心骨，还要花费大量的时间精力金钱去给我弟弟治病，她压力很大，也心事重重。

![弟弟脑出血](@/assets/images/2026/small-brother-ICH.jpg)

弟弟生病以后, 我们收到很多无私的帮助，几个姨妈， 姑妈，表堂兄弟姐妹们不少人来看望，
送钱送物，给我们带来温暖和希望。
老家的乡亲们，过去的同事同学，小区的邻居们，很多人已经很多年没见了，
纷纷出现在水滴筹的名单里，我只能默默的记在心里。
在大环境不怎么景气的情况下，有这么多人伸出援手，真的让我感动不已。

所幸的是 4 个月过去了，我弟弟已经拔掉的气管胃管，可以自己站起来了，也开始认识人，自己说话和吞咽了。
感谢上天，希望我弟弟能恢复的更加好。

## 2026 年展望

我弟弟的这个事情给我加强的警醒，我要更加注重自己的身体健康。我也是肥胖人群的一员，也经常熬夜，缺少锻炼。
小孩子也需要我更多的陪伴和教育。
新的一年，我希望自己能做到以下几点：

1. 坚持锻炼身体，每周至少锻炼三次；养成良好的饮食，作息习惯，保证充足的睡眠时间；2026 年底体重控制到 75 公斤以内。
2. 多陪伴家人，尤其是孩子的教育，培养孩子良好的习惯和性格；充分利用语言敏感期，让孩子在英语学习有一定成果。目标 200 个核心常用词和简单日常对话。
3. 定期安排家庭活动，如爬山，郊游，聚餐等，增进家庭成员之间的感情；
4. 工作上继续提升自己的技术能力和业务能力，争取在公司有更大的发展空间；
5. 学习 AI 相关技术，尝试将 AI 技术应用到工作和生活中。阅读更多的书籍，拓展自己的知识面；
6. 坚持输出，写博客和自媒体，分享自己的经验和见解。目标每周一篇博客或 vlog，希望单平台粉丝数达到 2000+。
7. 争取推进一个开源项目或自己的小产品，目标 1000 star 或者 1000 用户。

其中 1，2， 6，7 是我今年的重点目标，我也提供了量化目标，年底来检查。
希望 2026 年能成为我人生新的起点！
